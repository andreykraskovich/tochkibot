# tochkibot

Telegram-бот для ведения результатов матчей ЧМ 2026.

## Команды

| Команда | Описание |
|---|---|
| `/start` | Приветствие и список команд |
| `/myid` | Узнать свой Telegram ID |
| `/result Бразилия 2 Германия 1 A` | Добавить результат (группа опционально) |
| `/results` | Все результаты |
| `/standings` | Таблица групп |
| `/delete_result` | Удалить последний результат |

## Переменные окружения

```
BOT_TOKEN=токен от BotFather
DATABASE_URL=postgresql://...
ADMIN_IDS=123456789,987654321
```

`ADMIN_IDS` — Telegram ID тех, кто может вносить результаты (через запятую).  
Узнать свой ID: `/myid`.  
Если не задать — команды доступны всем.

## Деплой на Railway

1. Подключить репозиторий в Railway
2. Добавить переменные окружения
3. Подключить существующую PostgreSQL базу (или создать новую)
